# ☕ Coffee Prince | أمير القهوة

موقع تعريفي وتسويقي احترافي لمقهى متخصصة في تقديم القهوة المختصة والأجواء الفاخرة.

## 🎯 نظرة عامة

**Coffee Prince** هو موقع عصري وفاخر مخصص لعشاق القهوة الحقيقيين. يجمع الموقع بين التصميم الحديث والبساطة والوظائف المتقدمة، مع توفير تجربة مستخدم استثنائية على جميع الأجهزة.

## 📋 محتويات المشروع

### الصفحات الرئيسية:
- **الرئيسية (index.html)** - واجهة ترحيبية مبهرة مع معلومات سريعة وتنقل سلس
- **عن المشروع (about.html)** - قصة المشروع، القيم، الرؤية والرسالة
- **القائمة (menu.html)** - عرض تفاعلي للمنتجات مع إمكانية التصفية حسب الفئات
- **اتصل بنا (contact.html)** - نموذج اتصال وخريطة وجميع معلومات التواصل

### الملفات الرئيسية:
- `index.html` - الصفحة الرئيسية
- `about.html` - صفحة عن المشروع
- `menu.html` - قائمة المنتجات
- `contact.html` - صفحة التواصل
- `styles.css` - الأنماط الأساسية
- `pages.css` - الأنماط الإضافية للصفحات المختلفة
- `script.js` - الوظائف التفاعلية

## 📱 المحتوى

### الصفحة الرئيسية:
- صورة بطلة (Hero) جذابة
- معلومات سريعة (الموقع، أوقات العمل، رقم الهاتف)
- منتجات مختارة بصرية

### قائمة المنتجات:
- **مشروبات ساخنة**: إسبريسو، أمريكانو، كابتشينو، لاتيه، قهوة مقطرة، موكا
- **مشروبات باردة**: آيس كافيه، كولد برو، فراب كافيه
- **الطعام والحلويات**: كعك الشوكولاتة، كرواسون، كيك الجزر، سندويتشات
- عرض الأسعار والمكونات والوسوم لكل منتج

### معلومات التواصل:
- الموقع الجغرافي
- أرقام الهاتف
- البريد الإلكتروني
- أوقات العمل المفصلة
- روابط التواصل الاجتماعي
- نموذج اتصال كامل

## 🚀 الميزات التفاعلية

### JavaScript:
- **قائمة هاتف متحركة (Mobile Menu)**
- **تصفية المنتجات (Menu Filter)** - تصفية حسب الفئات
- **معالجة نموذج الاتصال** - التحقق من الصحة وإظهار الرسائل
- **التمرير السلس (Smooth Scroll)**
- **تفعيل الروابط النشطة (Active Links)**
- **رسوم متحركة عند التمرير (Scroll Animations)**
